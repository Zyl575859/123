# 📋 如何运行构建脚本

## 📍 文件位置

`build_apk_simple.py` 文件应该在项目根目录，和以下文件在同一目录：
- `main.py`
- `buildozer.spec`
- `ssh_tool_gui.py`
- `build_apk_simple.py` ← 这个文件

## 🚀 运行方法

### 方法1：使用批处理文件（最简单）⭐

**双击运行**：`运行构建脚本.bat`

会自动：
- 检查文件是否存在
- 检查Python是否安装
- 运行构建脚本

---

### 方法2：使用命令提示符

1. **打开命令提示符**
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目目录**
   ```bash
   cd C:\Users\Lenovo\Desktop\999999999
   ```

3. **运行脚本**
   ```bash
   python build_apk_simple.py
   ```

   或者：
   ```bash
   python3 build_apk_simple.py
   ```

---

### 方法3：直接在文件资源管理器中运行

1. **打开项目文件夹**
   - 找到 `build_apk_simple.py` 文件

2. **右键点击文件**
   - 选择 "打开方式"
   - 选择 "Python" 或 "Python Launcher"

---

## ❓ 常见问题

### Q: 提示"找不到文件"？
A: 
1. 确保在项目根目录运行
2. 检查文件是否存在：`dir build_apk_simple.py`

### Q: 提示"Python不是内部或外部命令"？
A: 
1. 安装Python：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 重启命令提示符

### Q: 如何确认当前目录？
A: 
在命令提示符中运行：
```bash
cd
dir build_apk_simple.py
```

如果显示文件，说明在正确目录。

---

## 💡 快速检查

运行以下命令检查一切是否就绪：

```bash
# 检查Python
python --version

# 检查文件
dir build_apk_simple.py

# 检查项目文件
dir main.py
dir buildozer.spec
```

如果都能找到，就可以运行了！
